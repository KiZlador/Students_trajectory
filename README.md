# 🎓 AI Teaching Assistant: Adaptive Learning Recommendation System

Интеллектуальная система рекомендаций учебных вопросов, адаптирующаяся под траекторию обучения студента. Проект демонстрирует полный цикл разработки ML-решения: от исследования и обучения state-of-the-art моделей до production-ready инференса с объяснимостью (Explainable AI).

## 🚀 Ключевые особенности

- **🧠 State-of-the-Art Архитектура**: SASRec (Self-Attentive Sequential Recommendation) на базе Transformer.
- **⚖️ Продвинутые функции потерь**: Реализация и сравнение BCE, gBCE (калибровка популярных элементов) и TRON (Sampled Softmax с динамическим негативным сэмплированием).
- **🔍 Объяснимый ИИ (XAI)**: Каждая рекомендация сопровождается объяснением, показывающим, какие именно вопросы из истории студента повлияли на выбор (на основе Attention Weights и косинусного сходства).
- **⚡ Быстрый инференс**: Векторный поиск ближайших соседей через **FAISS** (L2 нормализация = косинусное сходство).
- **🗂️ Автоматическая кластеризация**: Вопросы математически сгруппированы по темам с помощью KMeans. Оптимальное количество кластеров (k=5) выбрано через Elbow Method и Silhouette Score.

## 📊 Результаты и метрики

Модели обучались на подмножестве датасета **EdNet (KT1)** (50,000 пользователей, ~12,000 уникальных вопросов). После применения усиленной регуляризации (Dropout=0.5, Weight Decay=1e-4) лучшие результаты показала модель **TRON (Regularized)**:

| Модель | HR@10 | NDCG@10 | Статус |
| :--- | :---: | :---: | :--- |
| SASRec + BCE (Baseline) | 0.1561 | 0.0852 | Стабильный бейзлайн |
| SASRec + gSASRec (Reg) | 0.1647 | 0.1002 | Лучший Hit Rate |
| **SASRec + TRON (Reg)** | **0.1509** | **0.1184** | 🏆 **Лучшее ранжирование (NDCG)** |

*Примечание: Без регуляризации сложные модели (TRON/gSASRec) показывали сильное переобучение на валидации, что было успешно диагностировано и исправлено.*

- **Интерфейс**: Streamlit, Pandas, Matplotlib
- **Инфраструктура**: Python 3.12, Jupyter (Kaggle), WSL2

## ⚡ Как запустить локально

1. Клонируйте репозиторий и перейдите в папку проекта:
   ```bash
   cd recsys_project
   ```
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите веб-приложение:
   ```bash
   streamlit run app.py
   ```
5. Откройте браузер по адресу: `http://localhost:8501`

## 💡 Как это работает (XAI Demo)

1. Пользователь (или преподаватель) добавляет в "Траекторию" несколько решенных вопросов.
2. Система кодирует последовательность через SASRec, получая эмбеддинг текущей сессии.
3. Через FAISS находятся ближайшие векторы вопросов из словаря.
4. При нажатии **"💡 Почему этот вопрос?"**, система вычисляет косинусное сходство рекомендуемого вопроса с историей и анализирует Attention Weights последнего слоя трансформера, выдавая понятное текстовое объяснение и визуализацию влияния.
